Éléments HTML en ligne

HTML définit une longue liste de balises en ligne disponibles, dont une liste complète peut être trouvée sur le  Mozilla Developer Network.

  • Pour mettre du texte en gras , devraient utiliser <strong>.
  • Pour mettre du texte en italique, utilisez <em>.
  • Les abréviations, comme HTML, doivent utiliser  <abbr>, avec un attribut  de titre facultatif pour la phrase complète.
  • Les guillemets, comme — Mark Otto , utilisez  <cite>.
  • Le texte supprimé doit utiliser <del> et le texte inséré doit utiliser <ins>.
  • Le texte en exposant utilise <sup> et le texte en indiceutilise uses <sub>.

La plupart de ces éléments sont stylisés par les navigateurs avec peu de modifications de notre part.
